# Product list with cart solution

A responsive and interactive shopping cart application built with React, HTML5, and CSS. Users can add, remove, and adjust items in their cart, confirm orders through a modal, and start new orders easily. The interface adapts seamlessly to different screen sizes, with clear hover and focus states for all interactive elements.

## features

- Add items to the cart and remove them
- Increase/decrease the number of items in the cart
- See an order confirmation modal when they click "Confirm Order"
- Reset their selections when they click "Start New Order"
- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page
